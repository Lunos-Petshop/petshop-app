## Quick start
1. Clone
    git clone <repo-url>
    cd lunos-APP

2. Install dependencies
    npm install
    # or
    yarn

3. Environment
    Copy .env.example to .env and fill required values:
    cp .env.example .env

4. Run locally
    # For web / Expo / generic start
    npm start
    # For React Native (Android)
    npm run android
    # For React Native (iOS)
    npm run ios

5. Build / bundle
    npm run build
